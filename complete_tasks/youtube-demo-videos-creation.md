# YouTube Demo Videos Creation

## Task Description
Create short demo videos for the Trudify YouTube channel (@Trudify_com) showcasing the platform's core user journey.

## Video Plan

### Part 1: "Create a Task" (15 sec)
- Record in mobile mode on Mac
- Flow: Landing page → Login (Google OAuth) → Create task form → Submit
- **Sample Task:**
  - Title: Сглобяване на гардероб от IKEA
  - Description: Имам нов гардероб PAX от IKEA, трябва да се сглоби. Размери 2м x 2.5м. Всички части са налични.

**Voiceover (BG):**
```
Имаш нужда от помощ? С Trudify е лесно.
Влез в профила си, опиши задачата и я публикувай.
Само за секунди - готово!
```

### Part 2: "Accept Application" (15-20 sec)
- Show in-app notification from professional
- Accept the application
- Text overlay: "Професионалистът ще се свърже скоро"

**Voiceover (BG):**
```
Професионалисти ще видят задачата ти и ще кандидатстват.
Получаваш известие веднага.
Избери най-подходящия и приеми офертата.
Той ще се свърже с теб скоро!
```

## Text Overlays
| Timestamp | Text |

## Tools
- **Recording:** Mac screen recording (Cmd + Shift + 5) in mobile responsive mode
- **Voiceover:** NotebookLM by Google (AI-generated)
- **Text overlays:** CapCut or iMovie
