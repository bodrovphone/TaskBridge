# Video Guides for Professionals - YouTube Shorts Scripts

## Task Description
Create Bulgarian video scripts for YouTube Shorts targeting professionals (майстори) who want to use Trudify to find work. Each video should be 50-90 seconds (Video 4 can be slightly longer ~90-120 sec), suitable for ElevenLabs audio generation.

## Video List (4 Videos)

### Video 1: Registration & Profile Setup
**Duration:** 50-70 seconds

**Flow:**
- Register as professional (Google OAuth)
- Complete profile: add specialization, city, bio
- Add service categories (what services you offer)
- Upload avatar photo
- Show completed profile

**Key Message:** First impressions matter - a complete profile gets more clients

---

### Video 2: Browsing Available Tasks
**Duration:** 50-70 seconds

**Flow:**
- Go to "Browse Tasks" page
- Show all filters: category, city, budget range, date
- Apply filters to find relevant tasks
- View task details (description, budget, location, customer info)
- Show how to see customer's profile/rating

**Key Message:** Find tasks that match your skills and location

---

### Video 3: Applying to a Task (Submitting an Offer)
**Duration:** 50-70 seconds

**Flow:**
- From task detail page, click "Apply" / "Submit Offer"
- Fill in proposed price
- Add message to customer (why you're the right fit)
- Submit application
- Show confirmation

**Key Message:** Stand out with a competitive price and professional message

---

### Video 4: Full Task Journey - From Invitation to Completion
**Duration:** 90-120 seconds (longer video)

**Flow:**
- Receive notification about customer invitation (or accepted application)
- Open notification center
- View invitation/acceptance details
- Accept the invitation (if invitation flow)
- Task is now "In Progress"
- (Time skip) Work is done
- Go to "My Work" page
- Click "Mark as Complete"
- Confirmation dialog
- Submit completion
- Explain: customer confirms and leaves review

**Key Message:** Complete the full cycle - from accepting work to getting your review

---

## Acceptance Criteria
