## Campaign Launch Guidelines

### A. Pre-Launch Preparation

1. **Create Meta Business Account**
   - Go to business.facebook.com
   - Set up Business Manager
   - Add payment method (card)
   - Verify business (optional but recommended)

2. **Install Meta Pixel on Trudify**
   - Create pixel in Events Manager
   - Add pixel code to website (already done via Next.js)
   - Set up conversion events: SignUp, TaskCreated, ApplicationSubmitted

3. **Prepare Creative Assets**
   - Video in 9:16 format (Reels/Stories) - 1080x1920px
   - Video in 1:1 format (Feed) - 1080x1080px
   - Thumbnail images
   - Ad copy in Bulgarian (primary) + English

4. **Define Audiences**
   - Create Custom Audience from website visitors
   - Create Lookalike Audience from existing users
   - Save interest-based audiences for each professional category

### B. Campaign Setup (Step-by-Step)

1. **Go to Ads Manager** → Create Campaign

2. **Choose Campaign Objective:**
   - For awareness: "Video Views" or "Reach"
   - For signups: "Conversions" or "Lead Generation"
   - For app installs: "App Installs"
   - **Recommended start**: "Traffic" or "Conversions"

3. **Set Budget:**
   - Start with 5-10 €/day per ad set
   - Test for 3-5 days before scaling
   - Use "Campaign Budget Optimization" (CBO)

4. **Define Ad Set:**
   - **Location**: Bulgaria (or specific cities: София, Бургас, Пловдив, Варна)
   - **Age**: 25-55 (professionals), 25-65 (customers)
   - **Language**: Bulgarian
   - **Detailed Targeting**: Add keywords from list above
   - **Placements**: Automatic (or manual: Instagram Reels, Facebook Feed, Stories)

5. **Create Ad:**
   - Upload video
   - Write headline (max 40 chars): "Търсиш работа? Намери клиенти!"
   - Primary text (125 chars visible): Hook + value prop + CTA
   - CTA button: "Learn More" or "Sign Up"
   - Destination URL: trudify.com with UTM parameters

6. **Review & Publish**

### C. Post-Launch Optimization

1. **Monitor Daily (First Week):**
   - CTR (Click-through rate) - aim for >1%
   - CPC (Cost per click) - aim for <0.25 €
   - CPM (Cost per 1000 impressions)
   - Video watch time (aim for >50% at 3 sec)

2. **A/B Testing:**
   - Test different hooks (first 2 seconds)
   - Test different audiences
   - Test different CTAs
   - Run 2-3 variations per ad set

3. **Scaling Rules:**
   - If CTR >1.5% and CPC <0.15 € → increase budget 20%
   - If CTR <0.5% after 3 days → pause and revise creative
   - If CPA (cost per signup) is profitable → scale aggressively

4. **Retargeting:**
   - Create audience: "Visited site but didn't sign up"
   - Create audience: "Watched 50% of video"
   - Show different ad with stronger CTA
